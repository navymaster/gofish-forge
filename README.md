# Go Fish 🎣

Go Fish 是一个钓鱼体验加强的fabricmod。它从《星露谷物语》和《泰拉瑞亚》中汲取灵感，创造出一种令人愉悦的钓鱼体验，配有令人兴奋的新战利品、创意水生生物，以及享受钓鱼的动力。

此仓库是该mod的forge移植版，在deepseek-R1的帮助下快速进行的移植
因此不要对此仓库的代码质量有太高的期待，不过如果有bug可以提出来修复

fabric原版仓库：https://github.com/Draylar/go-fish

### 特性一览
---

🐟 新的有趣鱼类变体 - 钓鱼时在所有生物群落中找到主题水生生物！🦈

🎣 新的钓鱼竿和钓鱼饰品 ⌚

🥔 以水产食材🧂为基础的创意食品选择

💎 稀有战利品箱，里面有令人兴奋的战利品 🔮

🔥 下界和末地钓鱼，具有特定维度的钓鱼机制 🐠

---

你可以在CurseForge直接下载原版 [here](https://www.curseforge.com/minecraft/mc-mods/go-fish)
此移植版当前仅适配1.20.1和1.20.2，暂无更新计划
